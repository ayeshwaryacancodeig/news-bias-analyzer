Step-by-Step Guide to Host Your Website on GitHub Pages

  1. Create a GitHub Account: If you don't already have a GitHub account, sign up at github.com.

  2. Create a New Repository:

    • Go to your GitHub dashboard.
    • Click the "New" button to create a new repository.
    • Name the repository (e.g., news-bias-analyzer).
    • Make sure the repository is public.
    • Initialize the repository with a README file.

  3. Upload Your Files:

    • Once the repository is created, you can upload your index.html, styles.css, and scripts.js files.
    • Click on "Add file" and then "Upload files".
    • Drag and drop your files or select them manually.
    • Commit the changes.

  4. Enable GitHub Pages:

    • Go to the "Settings" tab of your repository.
    • Scroll down to the "GitHub Pages" section.
    • Under "Source", select the main branch and the /root directory.
    • Click "Save".

  5. Access Your Website:

    • After a few moments, GitHub Pages will provide a URL where your site is hosted.
    • The URL will typically be in the format https://<your-username>.github.io/<repository-name>/.
